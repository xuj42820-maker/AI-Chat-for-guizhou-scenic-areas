# 部署指南 — 免费上线你的项目

## 方案一：Render（推荐，完全免费）

### 第一步：注册 GitHub 账号
打开 https://github.com 注册一个账号（如果已有跳过）

### 第二步：安装 Git
下载安装 https://git-scm.com/downloads

### 第三步：把项目推到 GitHub

打开终端（PowerShell 或 Git Bash），执行：

```bash
cd d:\python期末作业\贵州旅游景点问答助手

# 配置 Git（只需第一次）
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"

# 初始化并提交
git init
git add .
git commit -m "贵州旅游景点智能问答助手"

# 在 GitHub 上创建仓库（打开 https://github.com/new，仓库名填 guizhou-travel-qa，不要勾选任何选项）
# 然后执行：
git remote add origin https://github.com/你的用户名/guizhou-travel-qa.git
git branch -M main
git push -u origin main
```

### 第四步：部署到 Render

1. 打开 https://render.com 注册账号（可以用 GitHub 账号直接登录）
2. 点击 **New +** → **Web Service**
3. 连接你的 GitHub 仓库 `guizhou-travel-qa`
4. 配置：
   - **Name**: `guizhou-travel-qa`
   - **Runtime**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn wsgi:app --bind 0.0.0.0:$PORT`
5. 点击 **Create Web Service**
6. 等待部署完成（约2-3分钟）
7. 部署成功后会给你一个网址，类似 `https://guizhou-travel-qa.onrender.com`

### 第五步：访问你的网站

打开上面的网址就能看到你的项目了！

## 方案二：Railway（新用户送$5免费额度）

### 步骤同上，到 Railway 平台操作：

1. 打开 https://railway.app 注册账号
2. 点击 **New Project** → **Deploy from GitHub repo**
3. 选择你的仓库
4. Railway 会自动检测 Python 项目并部署
5. 部署完成后会给你一个 `.up.railway.app` 的网址

**注意**：Railway 免费额度有限（$5用完要付费），适合短期展示。

## 常见问题

### Q: 部署后数据库是空的？
A: 正常的！`init_app()` 会在启动时自动导入数据。Render 免费版每次重启会重新初始化。

### Q: 图片加载不出来？
A: 图片在 `static/images/` 目录下，需要确保 GitHub 仓库里包含了这些图片。

### Q: 部署失败怎么办？
A: 检查 Render 的部署日志，常见原因：
- requirements.txt 缺少依赖（我们只用了 flask 和 gunicorn）
- Python 版本不对（runtime.txt 指定了 3.11.8）

### Q: 能绑定自己的域名吗？
A: Render 免费版不支持，付费版可以。Railway 支持。
