# 贵州旅游景点问答助手 — 项目开发计划

## 一、项目概述

### 1.1 项目名称
贵州旅游景点问答助手

### 1.2 项目目标
开发一个面向游客的贵州旅游智能问答 Web 应用，整合全省 135 个景点数据，提供 AI 问答、交互式地图、数据可视化、路线推荐等功能，帮助游客高效规划贵州之行。

### 1.3 技术选型

| 层级 | 技术方案 |
|------|----------|
| 后端框架 | Python 3.x + Flask |
| 数据库 | SQLite |
| 中文分词 | jieba |
| 前端渲染 | Jinja2 模板 + Vanilla JS |
| 数据可视化 | ECharts 5.4 |
| 地图服务 | 高德地图 JS API + REST API |
| CSS 架构 | Design Token 设计系统 |
| 跨域支持 | Flask-CORS |

### 1.4 项目周期

总工期约 3 周，分 5 个阶段推进。

| 优先级 | 项目 | 说明 |
|--------|------|------|
| 高 | 数据导入覆盖问题 | 每次启动 DELETE 重导，管理后台修改会丢失 |

| 中 | 问答系统接入 LLM | 当前为模板生成，可接入大模型提升回答质量 |
| 中 | 缺失景点图片补全 | 部分景点使用占位图 |
| 低 | 主题预览 / 导入导出 | 增强主题系统体验 |
| 低 | API 集成测试 | 补充 Flask test_client 端到端测试 |

# 地图:    http://localhost:5000/map
# 可视化:  http://localhost:5000/visualization
