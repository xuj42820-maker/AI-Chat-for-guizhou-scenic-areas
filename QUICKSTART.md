# 快速启动指南

## 🚀 快速开始

### 1. 环境准备

确保已安装 Python 3.8+ 和 pip：

```bash
python --version
pip --version
```

### 2. 安装依赖

```bash
cd 贵州旅游景点问答助手
pip install -r requirements.txt
```

### 3. 启动应用

```bash
python run.py
```

启动成功后会显示：
```
==================================================
贵州旅游景点问答助手启动
访问 http://localhost:5000 查看主页
访问 http://localhost:5000/visualization 查看可视化
访问 http://localhost:5000/map 查看地图
访问 http://localhost:5000/test-themes 查看主题测试
==================================================
```

### 4. 访问页面

- **主页**：http://localhost:5000
- **数据可视化**：http://localhost:5000/visualization
- **景点地图**：http://localhost:5000/map
- **主题测试**：http://localhost:5000/test-themes

## 🎨 主题系统

### 切换主题

1. 点击页面右上角的设置按钮 ⚙️
2. 在设置面板中选择主题：
   - 🌿 **探索模式**：明亮的旅游探索风格
   - 🤖 **AI科技模式**：深色科技感风格
   - 🏮 **国风贵州模式**：传统文化风格
3. 主题会自动保存，刷新页面后保持

### 主题特性

#### 🌿 探索模式（默认）
- 配色：绿色系 (#00B96B, #36CFC9)
- 风格：明亮、自然、清新
- 背景：贵州自然风景
- 地图：标准旅游地图

#### 🤖 AI科技模式
- 配色：青色/紫色系 (#00E5FF, #7C4DFF)
- 风格：深色、科技感、未来感
- 背景：数字科技背景
- 地图：深色科技地图
- 特效：粒子背景、发光效果

#### 🏮 国风贵州模式
- 配色：黔青色/朱砂红 (#2E7D6F, #C62828)
- 风格：传统、文化、水墨风
- 背景：水墨贵州背景
- 地图：手绘风格地图
- 特效：水墨效果、云纹装饰

## 🔧 开发说明

### 项目结构
```
贵州旅游景点问答助手/
├── app/                    # Flask应用
├── data/                   # 数据文件
├── scripts/                # 脚本工具
├── static/                 # 静态文件
│   ├── css/main.css       # 样式文件
│   ├── js/                # JavaScript
│   │   ├── themes/        # 主题配置
│   │   ├── animations.js  # 动效系统
│   │   └── main.js        # 主逻辑
│   └── images/            # 图片资源
└── templates/              # HTML模板
```

### 添加新主题
1. 创建主题配置文件：`static/js/themes/your_theme.js`
2. 在 `theme-engine.js` 中注册主题
3. 在 `base.html` 中添加脚本引用
4. 在设置面板中添加主题选项

### 自定义样式
所有样式使用CSS变量，修改变量即可改变主题：
```css
.my-element {
    background: var(--bg-surface);
    color: var(--text-primary);
    border: 1px solid var(--border-color);
}
```

## 🐛 常见问题

### Q: 主题切换不生效？
A: 
1. 检查浏览器控制台是否有错误
2. 清除localStorage后重试
3. 确保JavaScript文件正确加载

### Q: 地图不显示？
A:
1. 检查网络连接
2. 确认高德地图API密钥有效
3. 查看控制台错误信息

### Q: 动画卡顿？
A:
1. 检查设备性能
2. 在设置中降低动画速度
3. 系统会自动检测并禁用动画

### Q: 数据加载失败？
A:
1. 确认数据库文件存在
2. 检查API接口是否正常
3. 查看服务器日志
